# 小红书起号助手 (Xiaohongshu Account Booster)

小红书起号助手是一款自动化账号运营与内容采集工具，帮助用户高效管理小红书账号，进行内容采集、数据分析与自动发布。

## 功能列表

- **账号管理**：多账号管理与切换
- **内容采集**：自动采集小红书博主、作品、评论数据
- **数据分析**：采集后接入AI进行深度数据分析
- **自动发布**：批量发布内容到小红书
- **评论管理**：评论采集与互动分析

## 使用环境

- 操作系统：Windows 10 / Windows 11
- Python 3.11+（如需运行源码）
- 或直接运行编译后的 `XHSCollector.exe`

## 使用方法

### 方式一：运行编译版本
1. 从 Releases 下载最新版本的 `XHSCollector.exe`
2. 双击运行即可

### 方式二：源码运行
```bash
pip install -r requirements.txt
python 启动器.py
```
